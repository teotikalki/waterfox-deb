waterfox-deb

Deb packages and apt repository for Waterfox Web Browser for Ubuntu and its derivatives.

View waterfox-deb on GitHub

Installing

Method 1

Just install waterfox.deb package from here and apt repository will be automatically added.

Method 2

Run these commands to add this repostory to your APT sources list:

$ echo "deb https://hawkeye116477.github.io/waterfox-deb release main" | \ sudo tee /etc/apt/sources.list.d/waterfox-release-hawkeye116477.list

And the public GPG key:

$ curl https://hawkeye116477.github.io/waterfox-deb/hawkeye116477.key | sudo apt-key add -

You can use the packages after update your APT index:

$ sudo apt-get update

Then to install Waterfox, use Software Manager or type:

$ sudo apt-get install waterfox

Note: Language packs and KDE integration features are available as separate packages to install from apt repository!
